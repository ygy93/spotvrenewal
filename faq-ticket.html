    이용권 가격이 달라졌어요(안드로이드)

  • Google Play 스토어 정책이 새롭게 적용됨에 따라, 2022년 05월 31일부터 안드로이드 앱에서 이용권 구매 시, 아래와 같이 Google Play의 결제 수수료가 포함된 금액으로 결제가 진행됩니다. ▶ 안드로이드 - 베이직 이용권 : 8,690원 → 9,900원 - 프리미엄 이용권 : 16,390원 → 19,000원 *VAT 포함 단, 기존에 안드로이드에서 구매하신 이용권을 보유하셨다면, "현재 이용하고 계신 금액"으로 정기결제가 진행됩니다. WEB에서 구매하는 이용권은 기존과 가격이 동일하며, WEB에서 이용권을 구매한 후 APP에서도 서비스 이용이 가능합니다. ▶ WEB - 베이직 이용권 : 8,690원 - 프리미엄 이용권 : 16,390원 *VAT 포함

  • T우주
    Q. [T우주] 우주패스 all의 연결/해지는 어떻게 하나요?

  • SPOTV NOW X T우주 SPOTV NOW와 T우주의 제휴를 통해 "우주패스 all"에서 SPOTV NOW를 선택할 수 있게 되었어요!🎉 T우주 구독 서비스를 이용하고 계신다면, 우주패스 all에서 SPOTV NOW와 서비스 연결이 가능합니다! 서비스를 연결하신 계정으로 로그인해 주셔야 이용권의 확인 및 시청이 가능하기에 최초 서비스 연결 시 꼭 사용을 원하시는 계정으로 진행해 주세요. :) 📌해당 이용권은 우주패스 all의 제휴 서비스로, 연결과 해지는 모두 T우주에서만 가능합니다.

  • T우주
    Q. [T우주] 우주패스 all은 어떤 콘텐츠를 이용할 수 있나요?

  • 우주패스 all에서 SPOTV NOW 서비스를 연결하셨다면, "T우주 베이직 이용권"을 통해 SPOTV NOW 이용이 가능합니다! T우주 베이직 이용권은 아래와 같은 서비스를 이용하실 수 있어요.😃 ▶ T우주 베이직 이용권 → 모바일/PC 이용 가능 → 고화질(720p) → 모든 경기 LIVE / VOD → LIVE 채팅 서비스 → 모든 TV채널 📌T우주 베이직 이용권은 프리미엄 이용권으로 변경이 어렵습니다. 📌TV채널은 TV를 통해 시청하시는 상품이 아니며, 휴대폰/PC로 이용하실 수 있습니다. 📌기존 SPOTV NOW 이용권을 보유하셨다면 중복되지 않도록 이용권 만료 후에 선택해 주세요.

  • 네이버플러스 멤버십
    Q. [네이버플러스 멤버십] MY코리안리거가 무엇인가요?

  • 코리안리거를 함께 응원할 수 있는 [MY코리안리거] 서비스가 업데이트되었습니다!🎉 [네이버플러스 멤버십 스포츠 무제한 이용권]을 보유중이라면, 최대 5개의 MY코리안리거 선택할 수 있어요. 선택한 소속팀 경기 중 SPOTV NOW에서 중계되는 모든 LIVE 경기의 시청이 가능합니다!😃 MY코리안리거 경기를 시청하시면서, LIVE 채팅 서비스도 이용하실 수 있습니다. 같은 경기를 시청하고 계신 분들과 채팅을 통해 함께 경기를 즐겨보세요! 업데이트 된 MY코리안리거 기능과 함께, [TV채널] 탭 (SPOTV ON/ON2제외)과 SPOTV NOW에서 제공되는 모든 VOD는 기존과 동일하게 이용이 가능합니다. MY코리안리거를 선택하신 후, 자랑스러운 한국 선수들을 SPOTV NOW와 함께 응원해 주세요!

  • 네이버플러스 멤버십
    Q. [네이버플러스 멤버십] MY코리안리거는 어떻게 설정하나요?

  • MY코리안리거는 최대 5명의 선수까지 선택이 가능합니다. 📌이용권 기간 내에 1회만 변경이 가능하니 꼭 신중하게 선택해 주세요! 단, 최초 등록 시에는 다음 회차부터 변경이 가능합니다. 아래의 방법으로 MY코리안리거를 설정해 보세요!😃 [MY코리안리거 설정 방법] ▶ WEB 1. 우측 상단 초상화(프로필) 클릭 2. 마이페이지 클릭 3. MY코리안리거 설정 클릭 4. 추가 버튼 클릭 5. 응원하는 코리안리거(소속팀) 선택 후 저장 클릭 ▶ APP 1. 우측 상단 초상화(프로필) 터치 2. 더보기 터치 3. MY코리안리거 설정 4. + 버튼 터치 5. 응원하는 코리안리거(소속팀) 선택 후 저장 터치

[네이버플러스 멤버십]은 "네이버"에서 가입하실 수 있어요. 가입 후 디지털 콘텐츠에서 "SPOTV NOW"를 선택해 주세요! 현재 네이버 플러스 멤버십 SPOTV NOW 이용권은 총 3가지로 나누어져 있어요. ▶ 네이버플러스 멤버십 스포츠 무제한 이용권 → 모바일/PC 이용 가능 → 고화질(720p) → 선택한 MY코리안리거 LIVE 경기 / LIVE 채팅 서비스 → 모든 경기 VOD → TV채널 (ON/ON2제외)

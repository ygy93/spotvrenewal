    Q. 회원 탈퇴는 어떻게 하나요?

  • 탈퇴 시점부터 SPOTV NOW의 모든 콘텐츠를 이용하실 수 없으며, 이용권의 잔여기간과 상관없이 이용권이 즉시 소멸될 수 있으니 이 점 유의 부탁드릴게요! WEB에서 이용권을 구매하셨다면, 회원 탈퇴 시 정기결제가 자동으로 해지됩니다. 단, IOS 인앱결제 또는 Google 인앱결제를 통해 이용권을 구매하셨다면? 회원 탈퇴가 완료되어도 정기결제가 해지되지 않으니, 탈퇴 전 반드시 정기결제를 해지해 주세요! 자세한 정기결제 해지 방법은 [다음달 부터 결제되지 않도록 해주세요] FAQ 내용을 참고해 주세요.😊 회원 탈퇴 방법은 아래와 같아요. ▶ WEB 1. 로그인 후 우측 상단 초상화(프로필) 클릭 2. 회원 정보 클릭 3. 하단 [회원 탈퇴] 클릭

  • 비밀번호찾기
    Q. 아이디/비밀번호가 기억나지 않아요.

  • 카카오톡 / 페이스북 / 네이버 / APPLE 연동 계정의 아이디와 비밀번호를 잊으셨다면, 각 SNS 홈페이지에서 직접 아이디/비밀번호 찾기가 가능해요. 만약, SPOTV NOW에서 회원 가입 후 [이메일 계정]을 사용 중이시라면, 아래 방법을 통해 아이디와 비밀번호를 찾을 수 있어요.😊 [아이디 찾기] 1. 로그인 페이지에서 "이메일 찾기"(WEB) / "아이디 찾기"(APP) 선택 2. 가입한 휴대폰 번호를 입력 후 [이메일 찾기]를 선택 3. 휴대폰으로 전송된 "이메일" 확인 4. 전송된 이메일로 로그인 [비밀번호 찾기] 1. 로그인 페이지에서 "비밀번호 찾기" 선택 2. 가입한 이메일 계정을 입력 후 [임시 비밀번호 받기] 선택 3. 이메일로 전송된 "임시 비밀번호" 확인 4. "임시 비밀번호"로 로그인 5. 신규 비밀번호로 변경 후 사용

  • 연동계정
    Q. 비밀번호는 어떻게 바꾸나요?

  • 카카오톡 / 페이스북 / 네이버 / APPLE 연동 계정의 비밀번호를 변경하고 싶다면 각 SNS 홈페이지에서 직접 변경하실 수 있어요. SPOTV NOW에서 회원 가입한 이메일 계정의 비밀번호 변경은 아래 방법을 순서대로 따라해 보세요.😊 ▶ WEB 1. 우측 최상단 초상화(프로필) 클릭 2. 회원 정보 선택 3. "비밀번호 변경"을 클릭 4. 현재 비밀번호 / 신규 비밀번호 입력 5. 비밀번호 변경 완료 ▶ APP 1. 우측 상단 초상화(프로필) 터치 2. 상단 프로필의 [더보기] 터치 3. "비밀번호 변경" 터치 4. 현재 비밀번호 / 신규 비밀번호 입력 5. 비밀번호 변경 완료
